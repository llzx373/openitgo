## 2. #53 排版修复专项

- [ ] 超高图片：打开含超高插图的 EPUB（单图超过一屏）→ 单页/双页模式下图片缩放到一页内完整显示，不跨页拆分。
- [ ] 宽表格：技术书宽表格 → 单元格内长词折行，表格不溢出可视区。
- [ ] 长代码块：`pre` 块出现横向滚动条，而不是溢出或被截断。
- [ ] 复杂排版 EPUB：自带 `column-*`/`position: fixed` 的书不再出现多栏错乱/漂浮固定位；终端可见 `sanitize: stripped N conflicting style declaration(s)` 计数日志。

## 3. #52 菜单停放专项

- [ ] 电子书打开时点开菜单栏任一菜单 → 正文区变为当前主题阅读背景色（白/黑/羊皮纸），菜单文字清晰可读、不被正文穿透。
- [ ] 菜单关闭 → 正文立即恢复，无闪烁残留、无阅读位置丢失。
- [ ] 打开"快捷键一览"或加密压缩包密码对话框 → 同样停放，关闭恢复。
- [ ] 单页/双页/连续滚动三种模式下各验证一次。
