# OpenItGo

一款使用 Rust + egui + wgpu 构建的跨平台桌面漫画/小说阅读器，支持国漫（左→右）、日漫（右→左）和韩漫/Webtoon（长条从上到下）三种漫画阅读模式，EPUB/TXT/MOBI/AZW3/Markdown 电子书阅读，以及基于内嵌 libmpv 的视频/音频播放。

## 功能

### 阅读模式
- 国漫（左→右，LTR）
- 日漫（右→左，RTL）
- 韩漫/Webtoon（上→下，连续垂直滚动）
- **国漫与日漫支持双页模式**：一次并排显示两页
- **宽页检测**：明显横向长图在双页模式下自动单页全宽显示，阈值可在设置中调整
- **翻页动画**：可在设置中开启或关闭

### 支持的格式
- 本地图片文件夹
- `.zip` / `.cbz`
- `.cbr` / `.rar`
- `.pdf`
- `.epub`、`.txt`、`.mobi` / `.azw3`、`.md`（电子书）
- 视频/音频播放（内嵌 libmpv）：`.mp4` / `.mkv` / `.webm` / `.avi` / `.mov` 等视频格式，`.mp3` / `.flac` / `.aac` / `.m4a` / `.ogg` / `.wav` / `.opus` 等音频格式

### 电子书阅读
- 基于 `wry` 内嵌 WebView 渲染，支持单页、双页、连续滚动三种布局
- 可调整字体、字号、行间距、页边距与主题（白天 / 夜晚 / 羊皮纸）
- 自动按 EPUB 目录或 TXT/Markdown 标题分章，无章节标记时按字数切分虚拟章节
- **CSS Columns 分页**：单页/双页模式使用 CSS `columns` 对整章内容真实排版，再通过 `translateX` 平移内层 `#column-content` 显示当前 spread；配合相邻章节预加载与 CSS `transform` 过渡动画，避免漏边与重复行。
- **交互翻页**：点击屏幕左/右半边、滚轮（支持滚轮反转与水平方向）均可翻页；章末/章首自动跨章节切换
- **自适应重排**：改变字号/字体/边距或调整窗口大小后，自动重新测量并保留当前阅读位置
- **目录面板**：工具栏/菜单栏点击"目录"打开侧边栏章节列表，点击跳转
- **阅读进度**：底部状态栏显示当前章节标题、当前章 / 总章、当前页 / 总页数
- **历史与书签**：关闭应用或切换视图时自动保存当前章节与字符偏移，下次打开自动恢复；支持添加/删除电子书书签并跳转
- **书架混排**：电子书与漫画在同一书架显示，分"漫画 / 电子书"两个标签页（漫画页同时展示视频/音频条目），卡片带类型角标，点击电子书条目进入阅读

### 媒体播放
- 内嵌 libmpv 播放视频与音频，视频经 `CAOpenGLLayer` 渲染（视频层合成在 egui 透明 surface 之下，菜单与下拉框直接悬浮于画面之上），音频显示占位封面
- 工具栏控制：播放/暂停、±10s 跳转、0.5x/1x/1.5x/2x 倍速、字幕轨切换/关闭、音轨切换、音频输出设备选择、全屏；全屏时工具栏与进度条自动隐藏，鼠标移到顶部/底部边缘唤出
- 可拖拽进度条：拖动时按关键帧对齐预览，松手精确跳转；显示当前时间与总时长，右侧带音量滑块
- 音量控制：底栏滑块、滚轮（视频区滚动，每格 ±5%）、M 键静音；调整时画面右上角显示 OSD 反馈
- 偏好记忆：音量、倍速与输出设备全局记忆（写入设置，重启后生效）；保存的设备拔出时回退"自动"
- 书架集成：视频/音频文件与漫画、电子书同架展示与导入；视频封面由无头 mpv 截取 10% 处画面，音频取专辑封面
- 播放进度恢复：播放位置（毫秒）保存进历史记录，中途退出后自动续播，接近结尾时从头播放

媒体模式快捷键：

| 按键 | 行为 |
|---|---|
| Space | 播放 / 暂停 |
| ← / → | 快退 / 快进 5 秒 |
| J / L | 快退 / 快进 10 秒 |
| ↑ / ↓ | 音量 ±5（OSD 显示） |
| M | 静音 / 取消静音 |
| 1 / 2 / 3 / 4 | 0.5x / 1.0x / 1.5x / 2.0x 倍速 |
| V | 切换字幕轨 |
| F | 切换全屏 |
| Esc | 退出全屏，或返回书架 |

### 异步加载
- 文件解压、图片解码、PDF 渲染在独立后台线程进行，避免 UI 阻塞
- **当前页优先**：当前显示页使用高优先级队列，预加载页使用低优先级队列，确保打开或翻页时当前页尽快显示
- **缩略图优先**：先显示低分辨率缩略图，再替换为高清原图
- **基于大小的 LRU 缓存**：按设定的内存预算（100 MB - 16 GB，默认 1 GB）缓存已解码页面，自动淘汰最少使用的页面；上传 GPU 后释放 CPU 端像素
- **纹理压缩**：开启后解码线程将页面压缩为 DXT5/BC3，上传 GPU 前在 CPU 端解压（wgpu 后端），可按需启用

### 书架与元数据
- 卡片网格形式展示漫画、电子书与媒体文件的封面及标题，根据窗口宽度自动换行排列
- 封面缺失时自动在后台补生成；文件不存在的条目显示“已删除”标记
- 分"漫画 / 电子书"标签页展示（漫画页含视频/音频，卡片带类型角标），支持搜索、排序（最近阅读、标题、添加时间）
- 左键点击卡片打开漫画或电子书；右键菜单：打开、编辑标题、删除
- 递归扫描文件夹导入多个漫画文件/压缩包/电子书
- 阅读历史：单条删除、清空；同时保存 `comic_id` 与 `path`，匹配更稳定；电子书额外保存章节与字符偏移
- 书签：支持添加备注并编辑；电子书书签保存章节与字符偏移
- 一键“清理已删除”：移除书库中文件已经不存在的条目

### 界面控制
- **顶部菜单栏**：文件（打开文件夹、打开最近、返回书架）、视图（工具栏/状态栏/全屏/主题）、阅读（翻页、书签、模式、双页、缩放）、工具（设置）、帮助
- **顶部工具栏**：返回书架、模式切换、单/双页切换、缩放、上一页/下一页、页码跳转、添加书签、全屏、设置；电子书模式下显示目录、上一章/下一章、添加书签；可点击右上角 ✕ 隐藏，通过右键菜单或菜单栏恢复
- **底部进度条**：持续显示当前阅读进度，点击可跳转；鼠标悬停显示对应页缩略图预览；可点击右上角 ✕ 隐藏，通过右键菜单或菜单栏恢复
- **右键菜单**：下一页、上一页、首页、末页、添加书签、全屏、返回书架、显示/隐藏工具栏、显示/隐藏进度条
- **自动加载系统 CJK 字体**：确保中文界面正常显示

### 交互方式
| 操作 | 行为 |
|---|---|
| 拖拽文件/文件夹到窗口 | 打开漫画、电子书或媒体文件；若拖拽的是普通文件夹则递归扫描导入 |
| 拖拽文件/文件夹到 Dock 图标（macOS） | 应用未运行时也会启动并打开；文件夹会递归扫描导入 |
| 环境变量 `OPENITGO_OPEN` | 启动时自动打开指定路径的漫画/电子书（开发/测试用） |
| 滚轮 | 国漫/日漫翻页；Webtoon 垂直滚动；可在设置中反转滚轮方向 |
| Ctrl + 滚轮 | 缩放 |
| 双击页面 | 在原始尺寸（100%）与自动适应页面之间切换 |
| 拖拽页面 | 放大时平移 |
| 鼠标前进/后退键 | 翻页（日漫方向相反） |
| Ctrl + O | 打开文件夹并添加到书架 |

## 前置依赖

媒体播放功能依赖 libmpv，通过 Homebrew 安装：

```bash
brew install mpv
```

- 从源码构建与运行的机器需要安装，否则 `openitgo-media` 无法链接。
- `scripts/package-macos.sh` 打包出的 `.app` 已内嵌 libmpv 及其依赖，终端用户无需安装 mpv。

## 运行

```bash
cargo run -p openitgo-app
```

## 测试

```bash
cargo test --workspace
```

## 验证

提交前请运行完整检查：

```bash
cargo fmt --all
cargo check --workspace
cargo test --workspace
cargo clippy --workspace --all-targets -- -D warnings
```
