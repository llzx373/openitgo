# rustReader

一款使用 Rust + egui 构建的跨平台桌面漫画阅读器，支持国漫（左→右）、日漫（右→左）和韩漫/Webtoon（长条从上到下）三种阅读模式。

## 功能

### 阅读模式
- 国漫（左→右，LTR）
- 日漫（右→左，RTL）
- 韩漫/Webtoon（上→下，垂直滚动）
- **国漫与日漫支持双页模式**：一次并排显示两页

### 支持的格式
- 本地图片文件夹
- `.zip` / `.cbz`
- `.cbr` / `.rar`
- `.pdf`

### 异步加载
- 文件解压、图片解码、PDF 渲染在独立后台线程进行，避免 UI 阻塞
- **当前页优先**：当前显示页使用高优先级队列，预加载页使用低优先级队列，确保打开或翻页时当前页尽快显示
- **基于大小的 LRU 缓存**：按设定的内存预算（100 MB - 4 GB，默认 1 GB）缓存已解码页面，自动淘汰最少使用的页面

### 界面控制
- **顶部工具栏**：返回书架、模式切换、单/双页切换、缩放、页码跳转、添加书签、全屏、设置；可点击右上角 ✕ 隐藏，通过右键菜单恢复
- **底部状态栏**：页码、阅读模式、缩放比例、快捷键提示、页面导航条；可点击右上角 ✕ 隐藏，通过右键菜单恢复
- **右键菜单**：下一页、上一页、首页、末页、添加书签、全屏、返回书架、显示/隐藏工具栏、显示/隐藏状态栏
- **悬浮缩略图进度条**：鼠标移到阅读区底部时滑出，显示当前卷所有页面缩略图，点击跳转
- **自动加载系统 CJK 字体**：确保中文界面正常显示

### 交互方式
| 操作 | 行为 |
|---|---|
| 拖拽文件/文件夹到窗口 | 打开漫画 |
| 滚轮 | 国漫/日漫翻页；Webtoon 滚动 |
| 双击页面 | 切换全屏 |
| 拖拽页面 | 放大时平移 |
| Ctrl + O | 打开文件夹并添加到书架 |

### 键盘快捷键
| 按键 | 行为 |
|---|---|
| ← / → | 翻页（日漫方向相反） |
| PgDn / Space | 下一页 |
| PgUp | 上一页 |
| Home | 首页 |
| End | 末页 |
| + / - | 缩放 |
| 0 | 自动适应页面 |
| F11 / F | 切换全屏 |
| Esc | 退出全屏，或返回书架 |

## 运行

```bash
cargo run -p rust-reader-app
```

## 测试
