# rustReader

一款使用 Rust + egui 构建的跨平台桌面漫画阅读器，支持国漫（左→右）、日漫（右→左）和韩漫/Webtoon（长条从上到下）三种阅读模式。

## 功能

### 阅读模式
- 国漫（左→右，LTR）
- 日漫（右→左，RTL）
- 韩漫/Webtoon（上→下，垂直滚动）
- **国漫与日漫支持双页模式**：一次并排显示两页
- **宽页检测**：明显横向长图在双页模式下自动单页全宽显示，阈值可在设置中调整
- **翻页动画**：可在设置中开启或关闭

### 支持的格式
- 本地图片文件夹
- `.zip` / `.cbz`
- `.cbr` / `.rar`
- `.pdf`

### 异步加载
- 文件解压、图片解码、PDF 渲染在独立后台线程进行，避免 UI 阻塞
- **当前页优先**：当前显示页使用高优先级队列，预加载页使用低优先级队列，确保打开或翻页时当前页尽快显示
- **缩略图优先**：先显示低分辨率缩略图，再替换为高清原图
- **基于大小的 LRU 缓存**：按设定的内存预算（100 MB - 16 GB，默认 1 GB）缓存已解码页面，自动淘汰最少使用的页面；上传 GPU 后释放 CPU 端像素

### 书架与元数据
- 网格/卡片形式展示封面与标题
- 封面缺失时显示占位色块
- 支持搜索、排序（最近阅读、标题、添加时间）
- 右键菜单：打开、编辑标题、删除
- 递归扫描文件夹导入多个漫画文件/压缩包
- 阅读历史：单条删除、清空；同时保存 `comic_id` 与 `path`，匹配更稳定
- 书签：支持添加备注并编辑

### 界面控制
- **顶部工具栏**：返回书架、模式切换、单/双页切换、缩放、页码跳转、添加书签、全屏、设置；可点击右上角 ✕ 隐藏，通过右键菜单恢复
- **底部进度条**：持续显示当前阅读进度，点击可跳转；鼠标悬停显示对应页缩略图预览；可点击右上角 ✕ 隐藏，通过右键菜单恢复
- **右键菜单**：下一页、上一页、首页、末页、添加书签、全屏、返回书架、显示/隐藏工具栏、显示/隐藏进度条
- **自动加载系统 CJK 字体**：确保中文界面正常显示

### 交互方式
| 操作 | 行为 |
|---|---|
| 拖拽文件/文件夹到窗口 | 打开漫画；若拖拽的是普通文件夹则递归扫描导入 |
| 滚轮 | 国漫/日漫翻页；Webtoon 滚动；可在设置中反转滚轮方向 |
| 双击页面 | 切换全屏 |
| 拖拽页面 | 放大时平移 |
| 鼠标前进/后退键 | 翻页（日漫方向相反） |
| Ctrl + O | 打开文件夹并添加到书架 |

### 键盘快捷键
| 按键 | 行为 |
|---|---|
| ← / → | 翻页（日漫方向相反） |
| PgDn / Space | 下一页 |
| PgUp | 上一页 |
| Home | 首页 |
| End | 末页 |
| + / - | 缩放 |
| 0 | 自动适应页面 |
| F11 / F | 切换全屏 |
| Esc | 退出全屏，或返回书架 |

## 运行

```bash
cargo run -p rust-reader-app
```

## 测试

```bash
cargo test --workspace
```
